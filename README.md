#### Preface
* Backend (Phalcon, jQuery, Twitter Bootstrap)
* Frontend (Phalcon, jQuery, AngularJS, Twitter Bootstrap)

#### Implemented
* Staging development
* Service layer
* Phalcon Crypt / Decrypt cookies
* Phalcon + Angular JS token + cookies authentication

#### Requirements 
* Requires PHP 5.4
* Phalcon 1.3.3
* Memcache
* APC (APCu)
* PHP mcrypt library
* PHP PDO MySQL driver

#### Testing 
* PHP 5.5.9 (with FPM)
* MySQL 5.7.15
* libmemcached 1.0.8
